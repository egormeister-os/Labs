# Отчет по лабораторной работе

## Тема

Событийно-ориентированное графическое приложение `Reversi`, реализованное на Python с использованием библиотеки `pygame`.

## Соответствие заданию

- библиотека `pygame` используется как основа событийного цикла, обработки ввода и графического интерфейса
- настройки вынесены во внешние JSON-файлы
- при запуске отображается меню с обязательными пунктами
- реализованы звуки, фоновая музыка и анимация переворота фишек
- при новом рекорде открывается диалог ввода имени
- есть режим локальной игры на двоих с поворотом доски
- есть онлайн-режим на двоих по TCP
- код покрыт автотестами

## Архитектура

- `ReversiGame` отвечает за правила игры, проверку ходов, переворот фишек и определение победителя
- `ReversiController` управляет режимами, AI, рекордами и сетевым состоянием
- `PygameReversiApp` отвечает за отрисовку, меню, обработку событий и визуальную обратную связь
- `OnlineServer` и `OnlineClient` обеспечивают обмен JSON-сообщениями между игроками

## Сетевое взаимодействие

- серверная сторона запускается в режиме `host`
- клиентская сторона подключается в режиме `join`
- обмен выполняется JSON-сообщениями, разделенными символом новой строки
- сервер является авторитетным источником состояния партии и рассылает обновленное состояние после каждого принятого хода

## Тестирование

Тесты запускаются через `pytest`, покрытие контролируется `pytest-cov`. Основной упор сделан на:

- игровую логику
- AI
- загрузку конфигурации
- таблицу рекордов
- сетевой протокол
- контроллер приложения
- вспомогательные UI-функции
