# Reversi

Графическое приложение лабораторной работы по событийно-ориентированному программированию на Python с использованием `pygame`.

## Что реализовано

- главное меню: `начать игру`, `таблица рекордов`, `справка`, `выход`
- режимы игры:
  - против компьютера
  - тренировочный режим
  - локальная игра на двоих на одном экране без поворота доски
  - онлайн-игра на двоих по TCP (`host` / `join`)
- внешние конфигурации в JSON
- звуки и фоновая музыка с автоматическим fallback, если `pygame.mixer` недоступен
- анимация переворота фишек
- сохранение рекорда при результате выше первой строки таблицы
- документация в Markdown
- тесты `pytest` с требованием покрытия выше 90%

## Структура

- `main.py` — точка входа
- `reversi/model.py` — правила игры Reversi
- `reversi/controller.py` — управление режимами, рекордами и сетевой логикой
- `reversi/ui.py` — интерфейс на `pygame`
- `reversi/network.py` — онлайн-обмен сообщениями
- `reversi/audio.py` — генерация и проигрывание звуков
- `config/*.json` — настройки приложения и справка
- `tests/` — автоматические тесты

## Запуск

```bash
.venv/bin/python main.py
```

## Тесты и покрытие

```bash
.venv/bin/pytest
```

## Онлайн-режим

1. На первом компьютере выбрать `Начать игру -> Онлайн: host`.
2. На втором компьютере выбрать `Начать игру -> Онлайн: join`.
3. Ввести адрес в формате `host:port`, например `127.0.0.1:50007`.

## Конфигурация

Основные параметры находятся в [config/app.json](config/app.json), справка — в [config/help.json](config/help.json), таблица рекордов — в [config/leaderboard.json](config/leaderboard.json).
